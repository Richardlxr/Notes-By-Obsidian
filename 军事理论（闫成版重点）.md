# 第一章 中国国防


## 第一节、国防概述


### 国防的含义

国防是国家为抵御外来侵略与颠覆，捍卫国家主权、领土完整、维护国家安全、同意和发展，而进行的军事及与军事有关的政治、经济、可积、文化、教育、外交等方面的活动

### 现代国防的基本类型

1. **扩张型

2. **自卫型

3. **联盟型**：如 **“美日军事联盟”，“北约”

4. **中立型

### 现代国防的基本特征

1. **职能的双重性

2. **对抗的整体性

3. **目标的层次性

4. **手段的灵活性

### 古代的兵役制度

1. **秦汉时期的征兵制

2. **三国两晋南北朝时期的世兵制

3. **隋朝时期的府兵制

4. **宋朝的募兵制

5. **明朝的卫所兵役制

### 新时代防御性国防政策

1. **坚决捍卫国家主权、安全、发展利益

2. **坚持永不称霸、永不扩张、永不谋求势力范围

3. **贯彻落实新时代军事战略方阵

4. **坚持走中国特色强军之路

5. **服务构建人类命运共同体

## 第二节、国防建设


### 国防建设的领导体制

* **中共中央的国防领导职权
	
	1. 中国共产党是中国特色社会主义事业的领导核心。
	
	2. 中国人民解放军必须置于中国共产党的绝对领导之下，其最高领导权和指挥权属于中国共产党中央委员会和中央军事委员会。
	
* **全国人民代表大会及其常务委员会
	
	- **人大
		
		* **决定战争与和平问题（决定战争状态）
		
		* 制定有关国防方面的基本法律
		
		* 选举中央军事委员会主席 **（选举、罢免）
		
		* 审查和批准各类报告
		
		* 改变常委的决定 
		
	* **常委（有点类似）
		
		* 决定战争状态的宣布
		
		* 决定全国总动员或者局部动员
		
		* 制定有关国防方面的法律
		
		* 审查和批准各类报告
		
		* **监督中央军事委员会工作
		
		* 任免、规定衔级制度、决定授予荣誉称号.etc
		

* **国家主席 
	
	1. 根据人大决定，**宣布战争状态
	
	2. 根据人大决定，**发布动员令
	
	3. 公布人大制定的法律
	
	4. 授予勋章、荣誉称号
	
	5. 根据人大决定，批准和废除同外国缔结的国防相关条约

* **国务院 （领导和管理）
	
	1. **编制国防建设发展规划和计划
	
	2. 制定国防相关方阵、政策、行政法规
	
	3. 领导管理国防科研生产
	
	4. 管理国防军费、资产
	
	5. 退役军人安置、国防教育、征兵.ect

### 中国人民解放军

* 创建与1927.8.1**南昌起义**

* 是**中国共产党缔造和领导的人民军队、是我国武装力量的主体

* 中国人民解放军由**现役部队和预备役部队组成


### 中国人民武装警察部队

1. 武装警察部队的编制体制
	
	* **由党中央、中央军委集中统一领导，实行中央军委──武警部队──部队领导的指挥体制
	
	* 武装警察部队执行中国人民解放军的有关命令条例规定，但是==武装警察部队和中国人民解放军是并列关系，不是从属！
	
2. 武装警察部队的职能
	
	1. **海上维权执法**，维护国家主权尊严
	
	2. **预防和镇压敌对势力的破坏，应对紧急情况、维护社会治安
	
	3. **执行维稳和反恐任务，保证人民生命财产安全
	
	4. **防卫作战
