# 第一章 中国国防


## 国防概述


### 国防的含义

国防是国家为抵御外来侵略与颠覆，捍卫国家主权、领土完整、维护国家安全、同意和发展，而进行的军事及与军事有关的政治、经济、可积、文化、教育、外交等方面的活动

### 现代国防的基本类型

1. **扩张型

2. **自卫型

3. **联盟型**：如 **“美日军事联盟”，“北约”

4. **中立型

### 现代国防的基本特征

1. **职能的双重性

2. **对抗的整体性

3. **目标的层次性

4. **手段的灵活性

### 古代的兵役制度

1. **秦汉时期的征兵制

2. **三国两晋南北朝时期的世兵制

3. **隋朝时期的府兵制

4. **宋朝的募兵制

5. **明朝的卫所兵役制

### 新时代防御性国防政策

1. **坚决捍卫国家主权、安全、发展利益

2. **坚持永不称霸、永不扩张、永不谋求势力范围

3. **贯彻落实新时代军事战略方阵

4. **坚持走中国特色强军之路

5. **服务构建人类命运共同体
